# {metadata.title}

Trong hướng dẫn này, bạn sẽ học cách tạo và quản lý secret API keys trong Medusa Admin.

<Note>

Hướng dẫn này chủ yếu hữu ích cho các nhà phát triển và nhóm kỹ thuật xây dựng các tùy chỉnh cho ứng dụng Medusa.

</Note>

## Secret API Key là gì?

Secret API key là một token xác thực cho phép bạn truy cập các Admin API của Medusa. Bạn có thể truyền nó trong tiêu đề yêu cầu để gửi yêu cầu như một người dùng quản trị đã xác thực. Vì vậy, điều quan trọng là không tiết lộ secret API key trong mã phía máy khách hoặc các kho lưu trữ công khai.

Để tìm hiểu cách truyền secret API key trong tiêu đề yêu cầu, hãy tham khảo [Tham chiếu API](!api!/admin#2-api-token).

---

## Xem Secret API Keys

Để xem các secret API keys của người dùng hiện đang đăng nhập trong Medusa Admin, đi tới Cài đặt -> Secret API Keys.

Ở đây, bạn có thể thấy danh sách tất cả các secret API keys cho người dùng đã đăng nhập. Bạn cũng có thể tìm kiếm, lọc và sắp xếp các API keys để tìm key bạn đang tìm kiếm.

## Tạo Secret API Key

Khi bạn tạo một secret API key, bạn tạo nó cho người dùng hiện đang đăng nhập.

<Note>

Trước [Medusa v2.11.0], người dùng chỉ có thể có một secret API key đang hoạt động tại một thời điểm. Để tạo một secret API key mới, bạn phải [thu hồi](#revoke-secret-api-key) key hiện có trước. Nếu bạn không thể tạo nhiều secret API keys, hãy yêu cầu nhóm kỹ thuật của bạn [nâng cấp ứng dụng Medusa](!docs!/learn/update).

</Note>

Để tạo một secret API key mới cho người dùng hiện đang đăng nhập:

1. Đi tới Cài đặt -> Secret API Keys.
2. Nhấp vào nút Tạo trong tiêu đề phần chính.
3. Trong biểu mẫu mở ra, nhập tiêu đề của secret API key.
4. Khi bạn hoàn tất, nhấp vào nút Lưu.
5. Bạn sẽ nhận được một pop-up với secret API key. Sao chép nó và lưu trữ nó an toàn trước khi đóng pop-up, vì bạn sẽ không thể xem nó lại.

![Biểu mẫu tạo secret API key](https://res.cloudinary.com/dza7lstvk/image/upload/v1740495860/User%20Guide/Screenshot_2025-02-25_at_5.04.07_PM_u91vm5.png)

## Xem Chi tiết Secret API Key

Để xem chi tiết của một secret API key:

1. Đi tới Cài đặt -> Secret API Keys.
2. Nhấp vào một secret API key từ danh sách.

Điều này mở trang chi tiết của secret API key nơi bạn cũng có thể quản lý API key.

## Chỉnh sửa Secret API Key

Để chỉnh sửa một secret API key:

1. Đi tới trang chi tiết của secret API key.
2. Nhấp vào <InlineIcon Icon={EllipsisHorizontal} alt="three-dots" /> trong tiêu đề phần đầu tiên.
3. Chọn "Chỉnh sửa" từ menu thả xuống.
4. Trong cửa sổ bên mở ra, bạn có thể chỉnh sửa tiêu đề của secret API key.
5. Khi bạn hoàn tất, nhấp vào nút Lưu.

![Biểu mẫu chỉnh sửa secret API key](https://res.cloudinary.com/dza7lstvk/image/upload/v1740496144/User%20Guide/Screenshot_2025-02-25_at_5.08.53_PM_o4h3cn.png)

## Thu hồi Secret API Key

<Note type="warning">

Thu hồi một secret API key là không thể hoàn tác. Bạn không thể sử dụng key trong các yêu cầu sau khi thu hồi nó hoặc kích hoạt lại nó.

</Note>

Để thu hồi một secret API key:

1. Đi tới trang chi tiết của secret API key.
2. Nhấp vào <InlineIcon Icon={EllipsisHorizontal} alt="three-dots" /> trong tiêu đề phần đầu tiên.
3. Chọn "Thu hồi API key" từ menu thả xuống.
4. Xác nhận thu hồi API key bằng cách nhấp vào nút "Thu hồi API key" trong pop-up.

---

## Xóa Secret API Key

<Note type="warning">

Xóa một secret API key là không thể hoàn tác.

</Note>

Bạn chỉ có thể xóa một secret API key sau khi [thu hồi nó](#revoke-secret-api-key). Để xóa một secret API key:

1. Đi tới trang chi tiết của secret API key.
2. Nhấp vào <InlineIcon Icon={EllipsisHorizontal} alt="three-dots" /> trong tiêu đề phần đầu tiên.
3. Chọn "Xóa" từ menu thả xuống.
4. Xác nhận xóa API key bằng cách nhấp vào nút Xóa trong pop-up.
