# {metadata.title}

Trong hướng dẫn này, bạn sẽ học cách tạo và quản lý publishable API keys trong Medusa Admin.

<Note>

Hướng dẫn này chủ yếu hữu ích cho các nhà phát triển và nhóm kỹ thuật xây dựng các tùy chỉnh cho ứng dụng Medusa.

</Note>

## Publishable API Key là gì?

Publishable API key là một khóa khách hàng được phạm vi cho một hoặc nhiều kênh bán hàng. Khi bạn xây dựng một storefront, bạn phải truyền một publishable API key trong tiêu đề yêu cầu để chỉ định kênh bán hàng để phạm vi yêu cầu đến. Điều này ảnh hưởng đến các sản phẩm được truy xuất, số lượng tồn kho có sẵn và dữ liệu khác.

Để tìm hiểu cách truyền publishable API key trong tiêu đề yêu cầu, hãy tham khảo [Tham chiếu API](!api!/store#publishable-api-key).

## Xem Publishable API Keys

Để xem các publishable API keys trong Medusa Admin, đi tới Cài đặt -> Publishable API Keys.

Ở đây, bạn có thể thấy danh sách tất cả các publishable API keys bạn đã thiết lập trong cửa hàng của mình. Bạn cũng có thể tìm kiếm, lọc và sắp xếp các API keys để tìm key bạn đang tìm kiếm.

## Tạo Publishable API Key

Để tạo một publishable API key mới:

1. Đi tới Cài đặt -> Publishable API Keys.
2. Nhấp vào nút Tạo trong tiêu đề phần chính.
3. Trong biểu mẫu mở ra, nhập tiêu đề của publishable API key.
4. Khi bạn hoàn tất, nhấp vào nút Lưu.

![Biểu mẫu tạo publishable API key](https://res.cloudinary.com/dza7lstvk/image/upload/v1740490166/User%20Guide/Screenshot_2025-02-25_at_3.28.44_PM_aosbhw.png)

## Sao chép Publishable API Key

Để sao chép một publishable API key:

1. Đi tới trang chi tiết của publishable API key.
2. Trong phần đầu tiên, nhấp vào key đã rút gọn để sao chép nó vào clipboard của bạn.

![Sao chép publishable API key](https://res.cloudinary.com/dza7lstvk/image/upload/v1740500116/User%20Guide/Screenshot_2025-02-25_at_6.14.15_PM_muwq9e.png)

## Chỉnh sửa Publishable API Key

Để chỉnh sửa một publishable API key:

1. Đi tới trang chi tiết của publishable API key.
2. Nhấp vào <InlineIcon Icon={EllipsisHorizontal} alt="three-dots" /> trong tiêu đề phần đầu tiên.
3. Chọn "Chỉnh sửa" từ menu thả xuống.
4. Trong cửa sổ bên mở ra, bạn có thể chỉnh sửa tiêu đề của publishable API key.
5. Khi bạn hoàn tất, nhấp vào nút Lưu.

![Biểu mẫu chỉnh sửa publishable API key](https://res.cloudinary.com/dza7lstvk/image/upload/v1740490481/User%20Guide/Screenshot_2025-02-25_at_3.34.28_PM_hxppxj.png)

## Quản lý Kênh Bán hàng của Publishable API Key

Bạn có thể quản lý các kênh bán hàng của một publishable API key trong trang chi tiết của nó. Bằng cách thêm các kênh bán hàng vào publishable API key, nó trở thành một phần của phạm vi trong các yêu cầu được thực hiện với key.

### Thêm Kênh Bán hàng

Để thêm một kênh bán hàng vào một publishable API key:

1. Đi tới trang chi tiết của publishable API key.
2. Nhấp vào nút "Thêm" trong tiêu đề phần "Kênh Bán hàng".
3. Trong danh sách mở ra, chọn các kênh bán hàng bạn muốn thêm API key vào.
4. Khi bạn hoàn tất, nhấp vào nút Lưu.

![Biểu mẫu thêm kênh bán hàng](https://res.cloudinary.com/dza7lstvk/image/upload/v1740490636/User%20Guide/Screenshot_2025-02-25_at_3.37.05_PM_ontnzy.png)

### Xóa Kênh Bán hàng

Để xóa một kênh bán hàng khỏi một publishable API key:

1. Đi tới trang chi tiết của publishable API key.
2. Chọn hộp kiểm bên cạnh kênh bán hàng bạn muốn xóa trong phần "Kênh Bán hàng".
3. Nhấn <Kbd>R</Kbd> hoặc nhấp vào nút Xóa ở giữa dưới cùng của trang.
4. Xác nhận xóa các kênh bán hàng bằng cách nhấp vào nút Tiếp tục.

![Xóa kênh bán hàng khỏi publishable API key](https://res.cloudinary.com/dza7lstvk/image/upload/v1740490931/User%20Guide/Screenshot_2025-02-25_at_3.38.41_PM_pmlbqk.png)

## Thu hồi Publishable API Key

<Note type="warning">

Thu hồi một publishable API key là không thể hoàn tác. Bạn không thể sử dụng key trong các yêu cầu sau khi thu hồi nó hoặc kích hoạt lại nó.

</Note>

Để thu hồi một publishable API key:

1. Đi tới trang chi tiết của publishable API key.
2. Nhấp vào <InlineIcon Icon={EllipsisHorizontal} alt="three-dots" /> trong tiêu đề phần đầu tiên.
3. Chọn "Thu hồi API key" từ menu thả xuống.
4. Xác nhận thu hồi API key bằng cách nhấp vào nút "Thu hồi API key" trong pop-up.

---

## Xóa Publishable API Key

<Note type="warning">

Xóa một publishable API key là không thể hoàn tác.

</Note>

Bạn chỉ có thể xóa một publishable API key sau khi [thu hồi nó](#revoke-publishable-api-key). Để xóa một publishable API key:

1. Đi tới trang chi tiết của publishable API key.
2. Nhấp vào <InlineIcon Icon={EllipsisHorizontal} alt="three-dots" /> trong tiêu đề phần đầu tiên.
3. Chọn "Xóa" từ menu thả xuống.
4. Xác nhận xóa API key bằng cách nhấp vào nút Xóa trong pop-up.
