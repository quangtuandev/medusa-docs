# {metadata.title}

Medusa Admin đi kèm với hỗ trợ đa ngôn ngữ. Hỗ trợ đa ngôn ngữ chỉ dành cho giao diện Medusa Admin, không phải cho nội dung, chẳng hạn như mô tả sản phẩm.

Tài liệu này bao gồm danh sách đầy đủ các ngôn ngữ có sẵn, cả chính thức và của cộng đồng.

## Cách Thay đổi Ngôn ngữ

Để tìm hiểu cách thay đổi ngôn ngữ, hãy tham khảo hướng dẫn [Quản lý Hồ sơ](../../settings/profile/page.mdx).

---

## Hỗ trợ Ngôn ngữ Phải sang Trái (RTL)

<Note>

Hỗ trợ ngôn ngữ phải sang trái (RTL) có sẵn từ [Medusa Admin v2.11.0](https://github.com/medusajs/medusa/releases/tag/v2.11.0). Nếu bảng điều khiển của bạn không điều chỉnh theo ngôn ngữ RTL, hãy yêu cầu nhóm kỹ thuật của bạn [nâng cấp ứng dụng Medusa](!docs!/learn/update).

</Note>

Khi bạn thay đổi ngôn ngữ sang ngôn ngữ phải sang trái (RTL), chẳng hạn như tiếng Ả Rập hoặc tiếng Do Thái, giao diện Medusa Admin tự động điều chỉnh để hỗ trợ hướng văn bản RTL. Điều này bao gồm căn chỉnh văn bản và các phần tử UI phù hợp để cải thiện khả năng đọc và trải nghiệm người dùng.
