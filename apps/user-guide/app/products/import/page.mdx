## Nhập Sản phẩm từ File CSV

Nếu bạn đang di chuyển từ nền tảng thương mại khác, hoặc từ Medusa v1, bạn có thể nhập sản phẩm vào Medusa bằng file CSV.

Khi bạn nhập file CSV, Medusa sẽ tạo hoặc cập nhật sản phẩm dựa trên file CSV bạn tải lên.

### Định dạng File CSV

Trong biểu mẫu nhập, bạn có thể tải xuống file CSV mẫu để xem định dạng yêu cầu cho việc nhập.

#### Hàng trong File CSV

Mỗi hàng trong file CSV đại diện cho một biến thể sản phẩm để tạo hoặc cập nhật. Vì vậy, nếu một sản phẩm có nhiều biến thể, mỗi biến thể sẽ có hàng riêng của nó trong file CSV.

Ví dụ, một sản phẩm với hai biến thể (đỏ và xanh dương) sẽ có hai hàng trong file CSV, một cho biến thể đỏ và một cho biến thể xanh dương.

## Nhập Danh sách Sản phẩm

<Prerequisites
  items={[
    {
      text: "Một file CSV để nhập.",
      link: "#csv-file-format",
    }
  ]}
/>

Để nhập sản phẩm:

1. Đi tới trang Sản phẩm.
2. Nhấp vào nút Nhập trong tiêu đề phần chính.
3. Trong cửa sổ bên:
    1. Nếu bạn không chắc định dạng của file CSV phải như thế nào, bạn có thể tải xuống file CSV mẫu.
    2. Để tải lên file CSV bạn muốn nhập sản phẩm từ, kéo và thả file vào khu vực tải lên hoặc nhấp vào khu vực để mở trình chọn file.
    3. Sau khi bạn tải lên file CSV:
        1. Kiểm tra số lượng sản phẩm sẽ được tạo hoặc cập nhật.
        2. Để xóa file đã tải lên, nhấp vào biểu tượng <InlineIcon Icon={EllipsisHorizontal} alt="trash" />, và chọn Xóa từ menu thả xuống.
4. Khi bạn hoàn tất, nhấp vào nút Nhập.

<Note>

Nếu bạn gặp bất kỳ lỗi hoặc khó khăn nào, vui lòng liên hệ với nhóm hỗ trợ kỹ thuật của bạn vì điều này có thể là vấn đề trong [tích hợp lưu trữ](!resources!/infrastructure-modules/file) được sử dụng.

</Note>

Điều này bắt đầu quá trình nhập có thể mất một chút thời gian.

### Kiểm tra Trạng thái Nhập

Để kiểm tra xem việc nhập đang trong quá trình, đã hoàn thành, hoặc thất bại:

1. Nhấp vào biểu tượng <InlineIcon Icon={BellAlert} alt="bell" /> thông báo trong tiêu đề trang.
2. Tìm nhập sản phẩm, nên là mục mới nhất trong danh sách.

Văn bản trên mục nhập sẽ giải thích trạng thái hiện tại của việc nhập.

<Note>

Nếu việc nhập sản phẩm thất bại, vui lòng liên hệ với nhóm hỗ trợ kỹ thuật của bạn để kiểm tra nhật ký lỗi.

</Note>
